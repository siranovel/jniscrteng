jniscrteng
==========

## Requirement ##

libproxyjniに依存する(version 0.9のみ)  
(https://github.com/siranovel/libproxyjni.git)



## Usage ##
$ jniscrteng <js file> <upgrade-module-path>

## install ##
    git clone https://github.com/siranovel/jniscrteng.git  
    cd jniscrteng  
    rpm -ivh <rpmファイル名>
